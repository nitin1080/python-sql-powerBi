# Customer Behavior Analysis using Python, SQL & Power BI

## Project Overview
This project analyzes customer shopping behavior to uncover purchasing patterns, customer preferences, and business insights. The workflow includes data cleaning using Python, data analysis using SQL, and interactive dashboard creation using Power BI.

## Objectives
- Clean and preprocess customer shopping data.
- Perform SQL-based business analysis.
- Identify customer purchasing trends.
- Create interactive Power BI dashboards for visualization.
- Generate actionable business insights.

## Tech Stack
- Python (Pandas, NumPy, Matplotlib)
- MySQL
- SQL
- Power BI
- Jupyter Notebook

## Project Workflow
1. Data Collection
2. Data Cleaning & Preprocessing (Python)
3. Database Creation & SQL Analysis
4. Business Insight Generation
5. Dashboard Development (Power BI)

## Files Included

| File | Description |
|--------|-------------|
| customer_shopping_behavior.csv | Raw dataset |
| data_cleaning_with_python.ipynb | Data cleaning and preprocessing |
| customer_behavior_sql_queries.sql | SQL analysis queries |
| customer_behavior_dashboard.pbix | Interactive Power BI dashboard |
| Customer Shopping Behavior Analysis.pdf | Project report |
| Business Problem Document.pdf | Business requirements and objectives |

## How to Run
1. Clone the repository.
2. Open the Jupyter Notebook for data preprocessing.
3. Execute SQL queries in MySQL.
4. Open the `.pbix` file in Power BI Desktop to explore the dashboard.
